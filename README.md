# Supply-Chain-Logistics-Dashboard
📊 Supply Chain & Logistics Dashboard (Power BI)
1. 📌 Project Title 
Supply Chain Analytics Dashboard using Microsoft Power BI
A dynamic and interactive data visualization project designed to analyze supply chain performance, including suppliers, inventory, shipments, and customer data for better decision-making.
2. 📖  Purpose
This dashboard provides insights into key business metrics such as revenue, profit, order quantity, shipment performance, and delays. It helps stakeholders easily understand operations and identify areas for improvement.
3. 🛠 Tech Stack
The project was built using the following tools and technologies:
Microsoft Power BI Desktop – Dashboard creation and visualization
Power Query – Data cleaning and transformation (ETL)
DAX (Data Analysis Expressions) – Creating measures and KPIs
Data Modeling – Building relationships between multiple datasets
File Format – .pbix for development and .pdf for presentation
4. 📂 Data Source
The dataset includes supply chain data such as suppliers, inventory, shipments, and customers
Data was pre-processed and cleaned before analysis
Multiple tables were connected using relationships in model view
5. 🚀 Features 
🔹 Business Problem
Organizations often struggle to track supply chain performance due to scattered data across multiple sources.
🔹 Goal of the Dashboard
To provide a centralized and interactive dashboard for monitoring key supply chain metrics and improving decision-making.
🔹 Key Visuals
KPI Cards (Revenue, Profit, Orders, Shipments)
Inventory Analysis by Product
Shipment Quantity Overview
Delay Analysis by Carrier
Supplier Performance Insights
🔹 Business Impact & Insights
Helps identify delays in shipments
Tracks inventory levels efficiently
Improves supplier performance analysis
Enables better business decisions through data insights
